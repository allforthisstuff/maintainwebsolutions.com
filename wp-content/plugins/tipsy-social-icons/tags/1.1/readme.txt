== Description ==

A widget that makes it easy to display a variety of popular social icons along with a Facebook-style tooltips.  Icons provided by Komodo Media's Social Icon Pack and Jason Frame's jQuery Tipsy plugin for effects. Customizable via CSS.

Includes two icon sizes (16x16 and 32x32) and an optional hover effect.

Check out <a href="http://tommcfarlin.com/tipsy-social-icons">the project page</a> for more information and to follow plugin development.
		
== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Locate the widget in the Widget's admin area to begin using it.
