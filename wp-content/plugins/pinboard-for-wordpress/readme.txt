Pinboard for WordPress displays bookmarks from your Pinboard public feed  in your WordPress blog.

== Description ==

Pinboard for WordPress displays your latest pinboard.in public bookmarks in your WordPress blog.

**Features**

    * Widget ready
    * Uses Wordpress resources (no extra files needed)
    * Display user or global tags (optional)
    * Filter bookmarks by tag(s) (optional)
    
**Usage**

Drag the widget into your sidebar and configure.

If widgets are not your thing, use the following code to display your latest bookmarks:

`<?php pinboard_bookmarks("username"); ?>`

For more info (options, customization, etc.) visit [the plugin homepage](http://pinboard.in/resources/wordpress "Pinboard for Wordpress").

== Installation ==

Drop pinboad.php into /wp-content/plugins/ and activate the plugin.
